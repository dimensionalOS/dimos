# Language Interop Examples

Demonstrates controlling a dimos robot from non-Python languages.

## Usage

1. Start the robot (in another terminal):
   ```bash
   cd ../simplerobot
   python main.py
   ```

2. Run any language example:
   - [TypeScript](ts/)
   - [C++](cpp/)
   - [Lua](lua/)

3. (Optional) Monitor traffic with `lcmspy`
