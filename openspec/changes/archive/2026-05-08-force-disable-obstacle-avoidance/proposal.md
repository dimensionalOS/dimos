# 强制关闭避障

## 概要

为 Unitree Go2 的 Command Center 增加一个由操作员控制的强制移动覆盖能力，让人在手动驾驶机器人时可以临时关闭 Go2 避障。

## 动机

在狭窄环境、恢复动作或有人监督的测试场景中，Go2 避障可能会阻止操作员有意发出的手动移动指令。操作员需要一种明确、短时的覆盖方式，既能临时绕过避障，又不会修改正常机器人配置，也不会在键盘控制停止后让避障继续保持关闭。

## 范围

- 在 Command Center 中增加键盘手势，用于在手动移动期间请求强制移动覆盖。
- 从 Web 客户端通过 WebSocket 可视化模块发送覆盖请求。
- 将覆盖状态发布为 Go2 连接模块可消费的 typed stream。
- 覆盖生效期间临时关闭 Go2 避障。
- 在释放覆盖、停止移动、超时或 Go2 连接停止时，恢复配置中的避障状态。

## 非目标

- 不改变自主导航、探索或 agent skill 行为。
- 不持久化修改 Go2 避障配置。
- 不把该能力暴露为 LLM skill 或 MCP tool。
- 不允许在没有主动手动控制手势的情况下启用避障覆盖。

## 风险

该功能涉及机器人安全行为。它必须保持显式、临时、由操作员驱动。投入真实环境使用前，需要进行硬件验证。
